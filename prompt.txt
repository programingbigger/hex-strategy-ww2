# ゴール
- グラフビューによる、各ファイルの依存関係の可視化
    - 依存関係とは？
        - このプログラミングファイルAの入力の可視化、つまり1対nの関係性
- タグによる管理をしたい

# 設定方法
基本的には、md形式のファイルと設定は同じです。
以下には、ゴールを達成するためのチップスを載せておきます。この設定をすることによって、グラフィカルな依存関係の可視化が可能になります。
タグによる管理は、検索をする時に役立ちます

## 各ファイルの依存関係の設定方法
fileA⇨fileBのとき

### fileA
```md
これはfileAです。
```

### fileB
```md
このfileBは、[[fileA]]の依存者です。
```

## タグ付け方法
fileAに対して、「開発タグ」を付ける際は、以下のように設定
### fileA
```md
これはfileAです。
#fileA
```
